# 4P-GraphSearch

## To Compile/Run
1. `cmake . -B build` (on Windows native, use `cmake . -B build -G "MinGW Makefiles"`)
2. `cd build`
3. `make` (on Windows native, use `mingw32-make`) 
4. `./run-tests` or `./run-main`

## Important Notes:
- Do not modify the `tests/` folder.
